# 상품 재고 관리 시스템

상품 정보를 등록하고 재고 상태를 관리하는 Java 콘솔 프로그램입니다.

## 관리 정보

- 상품 ID
- 상품명
- 카테고리
- 가격
- 재고 수량
- 품절 여부

## 필수 기능

- [ ] 상품 추가
- [ ] 상품 삭제
- [ ] 상품 가격 수정
- [ ] 전체 상품 정보 출력
- [ ] 상품명으로 검색
- [ ] 카테고리별 상품 정보 출력
- [ ] 품절 여부 설정
- [ ] 가격 낮은 순으로 정렬해서 출력
- [ ] 재고가 n개 이하인 상품 정보 출력
- [ ] 카테고리별 상품 가격 평균 출력

## 사용 자료구조

- `ArrayList`
- `LinkedList`
- `HashMap`

## 추가 구현 기능

- 재고 수량 수정
- 상품 ID로 검색
- 품절 상품만 조회
